# 🧠 Focus Tracker

A minimal Python tool that measures your focus using keyboard activity.

## 🚀 Features

* Tracks keystrokes
* Detects idle time
* Calculates focus score
* Saves session history

## ▶️ Run

```bash
pip install -r requirements.txt
python main.py
```

Stop with `ESC` or `Ctrl+C`.

## 📊 Output Example

keystrokes: 300
focus_score: 82

## 📁 Data

All sessions are saved in `sessions.json`.

## 📌 Why this project?

Simple, unique, and demonstrates:

* Event handling
* File storage
* Real-time tracking
